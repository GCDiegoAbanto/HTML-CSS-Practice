The div tag

Definition and Usage

The <div> tag defines a division or a section in an HTML document.

The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

The <div> tag is easily styled by using the class or id attribute.

Any sort of content can be put inside the <div> tag!

Note: By default, browsers always place a line break before and after the <div> element.

Different elements

Div with borders

As stated previously, the divisions are not visible without styling. Because of this, there is a border on this one to get a better idea on how it works.
This is what the css code looks like:

#bordered{
 border: 5px solid black;
}

This code is contained inside a <style> tag in the head of this html file. The name #bordered is an id given to the div that contains this text, and it looks like this:

<div id="bordered">>

Div with color

This division has a colored background rather than a border.
This is what the css code looks like:

#colored{
 background-color: orange;
}

This code is contained inside the same <style> tag in the head of this html file. The name #colored is an id given to the div that contains this text, and it looks like this:

<div id="colored">>

Classes and tags

You can apply the same style to different elements using the class attribute, or by adding one style to all the instances of one element.
An example of a class

.bordered{
 border: 5px solid purple;
}

This would apply the border you can observe to all the tags with the attribute class = "bordered". A period (.) is used rather than a pound sign (#).
